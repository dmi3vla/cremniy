## Step 1 ‚ÄĒ FileNode animation primitives

**`file_node.h`** additions:
```cpp
Q_PROPERTY(qreal appearScale READ appearScale WRITE setAppearScale)

signals:
    void disappearFinished();

public:
    void playAppearAnimation(int durationMs = 300);
    void playDisappearAnimation(int durationMs = 250);
    qreal appearScale() const;
    void setAppearScale(qreal s);

private:
    qreal m_appearScale = 1.0;
```

**`file_node.cpp`** ‚ÄĒ `playAppearAnimation()`:
- Start: opacity=0, appearScale=0.5
- End: opacity=1, appearScale=1.0
- `QParallelAnimationGroup` with two `QPropertyAnimation` (on "opacity" and "appearScale")
- Easing: `QEasingCurve::OutBack`
- `setOpacity(0)` initially, then `setOpacity(1)` at end

**`file_node.cpp`** ‚ÄĒ `playDisappearAnimation()`:
- Start: opacity=1, appearScale=1.0
- End: opacity=0, appearScale=0.3
- Easing: `QEasingCurve::InCubic`
- `connect(finish, &disappearFinished)` ‚ÄĒ does NOT delete self

**`paint()`** ‚ÄĒ apply `m_appearScale` via `painter->scale(m_appearScale, m_appearScale)` before drawing.

## Step 2 ‚ÄĒ DependencyEdge animation primitives

**`dependency_edge.h`** additions:
```cpp
Q_PROPERTY(qreal edgeOpacity READ edgeOpacity WRITE setEdgeOpacity)
Q_PROPERTY(qreal drawProgress READ drawProgress WRITE setDrawProgress)

signals:
    void disappearFinished();

public:
    void playAppearAnimation(int durationMs = 300);
    void playDisappearAnimation(int durationMs = 250);

private:
    qreal m_edgeOpacity = 1.0;
    qreal m_drawProgress = 1.0;
```

**`dependency_edge.cpp`** ‚ÄĒ `paint()`:
- Apply `m_edgeOpacity` to pen/brush alpha
- If `m_drawProgress < 1.0`: draw `curvePath().pointAtPercent(m_drawProgress)` as endpoint instead of full path

**Appear**: `QParallelAnimationGroup` ‚ÄĒ edgeOpacity 0‚Üí1, drawProgress 0‚Üí1. Easing: `OutCubic`.
**Disappear**: edgeOpacity 1‚Üí0. Easing: `InCubic`. Emit `disappearFinished`.

## Step 3 ‚ÄĒ CanvasLayout::animateNodesToPositions

**New public method**:
```cpp
void CanvasLayout::animateNodesToPositions(
    const QMap<QString, QPointF>& targetPositions,
    QMap<QString, FileNode*>& nodes,
    int durationMs = 400);
```

**Implementation**:
- Create `QParallelAnimationGroup`
- For each node in `nodes` that has a key in `targetPositions`:
  - `QPropertyAnimation(node, "pos", parent)` ‚ÄĒ from `node->pos()` to `targetPositions[path]`
  - Easing: `QEasingCurve::InOutCubic`
- Add all animations to group, `group->start(QAbstractAnimation::DeleteWhenStopped)`

**Refactor `layoutNodesRadial()`**:
- Keep `computeTargets()` call
- Replace direct `setPos()` loop with `animateNodesToPositions(targets, m_nodes)`

## Step 4 ‚ÄĒ CanvasTab::diffGraphUpdate

**New method**:
```cpp
void CanvasTab::diffGraphUpdate(const DependencyGraph& oldGraph, const DependencyGraph& newGraph);
```

**Logic**:
1. **added** = `newGraph.allFiles - oldGraph.allFiles`
   - Create FileNode, compute position (near parent dir, not full layout), `addItem`, `playAppearAnimation()`
   - Create DependencyEdges for added file's includes, `playAppearAnimation()`

2. **removed** = `oldGraph.allFiles - newGraph.allFiles`
   - `node->playDisappearAnimation()` ‚Üí `connect(disappearFinished, scene->removeItem + deleteLater)`
   - Remove connected edges similarly

3. **changed** = files where `dependencies[]` list differs
   - Diff edges: added edges ‚Üí create + `playAppearAnimation()`; removed edges ‚Üí `playDisappearAnimation()` + remove

4. **Unchanged nodes**: position NOT touched (no jump)

**Replace handler** in `setFile()`:
```cpp
connect(m_parser, &DependencyParser::graphUpdated, this,
    [this](DependencyGraph graph) {
        diffGraphUpdate(m_currentGraph, graph);
        m_currentGraph = graph;
    });
```
